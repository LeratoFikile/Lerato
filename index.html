🔹 My Role & Ownership

Role: Backend Lead

Project Overview

QhaweLami (“My Hero” in isiZulu) is a gamified web and mobile platform designed to combat Gender-Based Violence and Femicide (GBVF) in South Africa through education, community engagement, and digital safety tools.

Problem Statement

Key Features

  • Custom Wall of Honour feature implemented independently

TaskServices: Intelligent Task Scheduler

NDEV 743 Major Assignment

Role: Individual Developer

Status: Completed ✓

Project Overview

TaskServices is an intelligent task management system built to demonstrate deep understanding of fundamental data structures. The system efficiently manages tasks by leveraging arrays, stacks, queues, double linked lists, binary trees, and recursive algorithms for different operations.

Core Functionality

  • Add, delete, and search tasks efficiently
  • Manage priorities and deadlines
  • Undo/Redo operations using stacks
  • Priority-based task execution
  • Data structure browsing and exploration

Data Structures Implemented

Arrays Stacks Queues Double Linked Lists Binary Trees Recursion

Technical Stack

Java Spring Boot HTML5 CSS3 JavaScript

Key Features

  • Task Management: Create, update, delete tasks with attributes (ID, title, priority, deadline, status)
  • Search & Navigation: Multi-criteria search across different data structure implementations
  • Undo/Redo: Full operation history with stack-based tracking
  • Priority Processing: Execute tasks in order of priority
  • Web Interface: Simple, clean UI for task management
